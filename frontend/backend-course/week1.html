Week 1: Backend Foundations

📌 What is Backend?

Backend is the core part of any application that runs behind the scenes. It handles data processing, authentication, and communication with the database.

Whenever a user performs an action like login, signup, or data fetch, the backend processes that request and returns a response.

🌐 How Web Applications Work

  • User interacts with frontend
  • Frontend sends request to backend
  • Backend processes logic
  • Database is queried
  • Response is returned to frontend

⚙️ Backend Responsibilities

  • Data storage & management
  • User authentication
  • API handling
  • Business logic execution
  • Security implementation

🧠 Frontend vs Backend

Frontend Backend
User Interface Server Logic
HTML, CSS, JS Node.js, Database
User Interaction Data Processing

🧰 Backend Tech Stack

  • Node.js (Runtime)
  • Express.js (Framework)
  • MongoDB (Database)
  • JWT (Authentication)
