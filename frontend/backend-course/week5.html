Week 5: Authentication & Advanced APIs

📌 What is Authentication?

Authentication is the process of verifying a user's identity. It ensures that only valid users can access protected resources.

Example: Login system where user enters email & password.

🔐 Authentication Flow

  • User enters credentials
  • Backend verifies data
  • Password is checked
  • Token is generated
  • User gets access

🔑 Password Hashing (bcrypt)

Passwords should never be stored as plain text. We use hashing to secure them. 

const bcrypt = require("bcrypt");

const hashedPassword = await bcrypt.hash("123456", 10);

✔️ Password Verification

const isMatch = await bcrypt.compare("123456", user.password);

🎟️ What is JWT?

JWT (JSON Web Token) is used to maintain user sessions securely. 

const jwt = require("jsonwebtoken");

const token = jwt.sign({ id: user._id }, "secretKey");

🛡️ Protected Routes

const authMiddleware = (req, res, next) => {
  const token = req.headers.authorization;

  if (!token) return res.send("Access denied");

  try {
    const decoded = jwt.verify(token, "secretKey");
    req.user = decoded;
    next();
  } catch {
    res.send("Invalid token");
  }
};

🔗 Apply Middleware

app.get("/profile", authMiddleware, (req, res) => {
  res.send("User Profile");
});
