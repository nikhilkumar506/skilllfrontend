Week 4: MongoDB – Database & CRUD Operations

📌 What is a Database?

A database is used to store and manage application data. Without a database, your application cannot save user information.

Example: User login data, posts, comments — all are stored in a database.

🍃 What is MongoDB?

MongoDB is a NoSQL database that stores data in JSON-like format. It is flexible, scalable, and widely used in modern applications.

  • No tables → Uses collections
  • No rows → Uses documents
  • Data stored as JSON

📊 MongoDB Structure

  • Database → Main container
  • Collection → Group of data
  • Document → Single record
{
  name: "akhilesh",
  email: "akhilesh@gmail.com"
}

⚙️ Setup MongoDB

Install mongoose:

npm install mongoose

Connect database:

const mongoose = require("mongoose");

mongoose.connect("mongodb://127.0.0.1:27017/zertix")
.then(() => console.log("DB Connected"))
.catch(err => console.log(err));

📦 Schema & Model

const userSchema = new mongoose.Schema({
  name: String,
  email: String
});

const User = mongoose.model("User", userSchema);

🔁 CRUD Operations

Create:

await User.create({ name: "akhilesh", email: "test@gmail.com" });

Read:

const users = await User.find();

Update:

await User.updateOne({ name: "akhilesh" }, { email: "new@gmail.com" });

Delete:

await User.deleteOne({ name: "akhilesh" });

🔗 Connect MongoDB with Express

app.post("/add-user", async (req, res) => {
  const user = await User.create(req.body);
  res.json(user);
});

🧠 Real-World Example
