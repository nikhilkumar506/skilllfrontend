Week 3: Express.js – Building Real APIs

📌 What is Express.js?

Express.js is a lightweight web framework for Node.js that helps you build APIs and backend applications easily.

Instead of writing complex server code using the http module, Express simplifies everything using routes and middleware.

🚀 Why Use Express?

  • Simple and fast
  • Easy routing system
  • Middleware support
  • Used in real-world projects

⚙️ Setup Express

npm init -y
npm install express

Create a file: server.js

🌐 Create Your First Express Server

const express = require("express");
const app = express();

app.get("/", (req, res) => {
  res.send("Hello from Express");
});

app.listen(3000, () => {
  console.log("Server running on port 3000");
});

🛣️ Routing in Express

app.get("/users", (req, res) => {
  res.send("All users");
});

app.post("/login", (req, res) => {
  res.send("User login");
});

GET → fetch data
POST → send data

🧠 Middleware Concept

Middleware runs between request and response. It is used for validation, logging, and security. 

app.use((req, res, next) => {
  console.log("Request received");
  next();
});

📦 Handling JSON Data

app.use(express.json());

app.post("/data", (req, res) => {
  console.log(req.body);
  res.send("Data received");
});

🚀 Mini Project: Simple API

Create a basic user API:

let users = [];

app.post("/add-user", (req, res) => {
  users.push(req.body);
  res.send("User added");
});

app.get("/users", (req, res) => {
  res.json(users);
});

🧠 Real-World Use

Every app like Instagram, Netflix, or Amazon uses APIs built with frameworks like Express to handle data.
